# Clean Browsing v1.6.2

**Release type:** Patch
**Extension version:** 1.6.2
**Target:** Firefox (Manifest V2, gecko min 109.0)

## What's new

### Add Widget modal redesign

The **Add widget** dialog now uses the same fixed-size shell as the main
Settings dialog (`min(90vh, 760px)` tall, `max-w-4xl` wide). Instead of a
compact two-column card list that grew unwieldy as more widgets were added,
the widget catalog now renders in a scrollable **three-column grid** inside
a styled header + body layout. On narrow viewports (below 640 px) the grid
collapses to two columns so the cards stay readable.

No tabs and no footer — the interaction is unchanged: click a widget card
to drop it onto the grid and close the dialog.

### Refresh on page load

Widgets that fetch live data now force-refresh on every page load — opening
or reloading a new tab counts as a refresh, in addition to the existing
hover-refresh button and the auto-refresh interval.

- **Weather** — previously served the cached forecast on reload if it was
  within the refresh interval. Now it refetches on every page load via a
  one-shot `onMount` call, separate from the cache-aware interval logic.
  The cached forecast still renders instantly while the fresh fetch
  completes in the background.
- **Ping Monitor** — previously waited for the first interval tick before
  running its first check after page load. Now it runs a check immediately
  on mount, so the latest status shows up without delay.
- **Stock** and **Stock Watchlist** — already refreshed on mount via their
  existing symbol-change effects; no behavior change.

A defensive `if (loading) return;` guard was added to the Weather widget's
`refresh()` so the new page-load call and the existing reactive-effect call
can coexist without issuing two parallel fetches on first mount.

## Breaking changes

None. Existing layouts, settings, and widget instances carry over unchanged
from v1.6.1.

## Installation

### From a GitHub release artifact

1. Download `clean_browsing-1.6.2.zip` (or the signed `.xpi`) from the
   [v1.6.2 release page](https://github.com/dmeim/clean-browsing/releases/tag/v1.6.2).
2. In Firefox, open `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-on...** and pick the `manifest.json` inside
   the extracted zip (or the `.xpi` file directly).
4. Open a new tab to start using Clean Browsing.

### From source

```bash
git clone https://github.com/dmeim/clean-browsing.git
cd clean-browsing
git checkout v1.6.2
npm install
npm run build
```

Then load `dist/manifest.json` via `about:debugging`.
