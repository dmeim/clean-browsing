# Clean Browsing v1.4.1

**Release type:** Patch
**Extension version:** 1.4.1
**Target:** Firefox (Manifest V2, gecko min 109.0)

## What's new

### Notes — "Clear completed" action

Hovering over a Notes widget that has checked-off task items now reveals a
**Clear completed** button in the top-right corner. Clicking it removes all
`[x]` lines from the note in one action. The button only appears on hover
and hides when no tasks are completed, keeping the widget clean and
distraction-free.

## Breaking changes

None. Existing layouts, settings, and widget instances carry over unchanged
from v1.4.0.

## Installation

### From a GitHub release artifact

1. Download `clean_browsing-1.4.1.zip` (or the signed `.xpi`) from the
   [v1.4.1 release page](https://github.com/dmeim/clean-browsing/releases/tag/v1.4.1).
2. In Firefox, open `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-on…** and pick the `manifest.json` inside
   the extracted zip (or the `.xpi` file directly).
4. Open a new tab to start using Clean Browsing.

### From source

```bash
git clone https://github.com/dmeim/clean-browsing.git
cd clean-browsing
git checkout v1.4.1
npm install
npm run build
```

Then load `dist/manifest.json` via `about:debugging`.
